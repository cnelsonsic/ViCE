Architecture¶

As can be seen from the architecture stack diagram above, ViCE’s core is composed of the plugin framework and database layer. Item, Action, and other, yet to be developed plugin types sit directly on these components. While not yet started, other features and abstractions suitable for designers and players will then be built on top of these.
